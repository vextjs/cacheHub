# v2.2.4 变更详情

> **发布日期**: 2026-06-11
> **类型**: Patch release
> **前置版本**: v2.2.3

---

## Added

- 新增 `scripts/memory-leak-pressure.mjs`，提供公开 API 方法级内存压力探针。
- 新增 `npm run test:memory`，使用 `node --expose-gc` 压测发布 ESM 产物。
- 新增 `npm run test:memory:build`，用于源码仓库发版前先构建再执行内存压力探针。

---

## Changed

- 发布包 `files` 增加 `scripts/memory-leak-pressure.mjs`，确保发布包中的 `test:memory` 命令可执行。
- `prepublishOnly` 增加 `npm run test:memory`，让真实 `npm publish` 前也执行方法级内存压力探针。
- Profile 测试规范补充内存压力测试路线，以及 Redis `clear()` 只能在 fake Redis 场景覆盖的安全边界。

---

## Compatibility

- 无 breaking changes。
- 不修改 `CacheLike`、exports、运行时代码或公开 API。
- `dependencies` 继续为空；`ioredis` 仍为 optional peer dependency。
